## About VLC Media Player : 
VLC Media Player is developed by the VideoLAN project. It is a multimedia player that supports almost all audio and video formats without requiring additional codecs. This makes it very different from many other media players which depend on external plugins.
One interesting fact is that VLC was originally created by students. Over time, it evolved into a global open-source project with contributions from developers all around the world.

## Why VLC Was Created : 
Before VLC, users often faced problems like unsupported formats and paid software limitations. Many media players required users to install extra codecs or buy licenses.
The main goal of VLC was to solve these issues by creating a simple, powerful, and completely free media player. Instead of keeping it private, the developers decided to release it as open source. This decision allowed others to contribute and improve the software continuously.

## License and Freedom
VLC is released under the GNU General Public License (GPL). This license ensures that users have the freedom to:
	•	Use the software freely
	•	Study how it works
	•	Modify it according to their needs
	•	Share it with others
An important rule of GPL is that any modified version must also be shared under the same license. This prevents companies from taking open-source work and making it completely closed.
